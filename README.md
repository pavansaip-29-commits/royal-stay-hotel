# Royal Stay Hotel Website

## 📌 Project Description
Royal Stay Hotel is a responsive hotel website developed using HTML5, CSS3, and JavaScript. It provides information about the hotel, available rooms, hotel packages, gallery, booking, and contact details.

## 🚀 Features
- Responsive Design
- Home Page
- About Us Page
- Rooms Page
- Packages Page
- Gallery Page
- Booking Form with JavaScript Validation
- Contact Page
- Simple Navigation Menu
- Basic SEO (Title, Meta Description)

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Install the **Live Server** extension (optional).
4. Open `index.html`.
5. Right-click and choose **Open with Live Server**, or open `index.html` directly in your browser.
